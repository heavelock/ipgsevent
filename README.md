IPGSEventCreator
===============================

version number: 0.0.1
author: Damian Kula

Overview
--------

IPGSEventCreator creates .ics files for IPGS seminars

Installation / Usage
--------------------

To install use pip:

    $ pip install ipgsevent


Or clone the repo:

    $ git clone https://github.com/heavelock/ipgsevent.git
    $ python setup.py install
